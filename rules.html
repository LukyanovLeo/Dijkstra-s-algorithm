HOW TO PLAY

Schulte table is the nice way to improve your peripheral vision skills.
Also it helps you to be more attentive and observant.

Schulte table is the square table, consisted of set of numbers,
which are written in the fields in random manner
You just need to find all the numbers from 1 to the last one in ascending order
When you've found next number you must click on it buy left mouse button.
There is a tip on the screen below so if you've forgotten the next number you can glance over it

When all the work is done you'll see the message with suggestion to save your result in seconds.
If you said "No." the application will offer you to play again.
